# Dropout Regularization in Neural Networks using MNIST

##  Project Overview
This project demonstrates how dropout regularization improves the generalisation performance of neural networks. Using the MNIST handwritten digit dataset, the study compares models trained with and without dropout to highlight the impact of overfitting and regularisation.

The tutorial combines theoretical explanation, experimental analysis, and visual interpretation to provide a comprehensive understanding of dropout.

---

##  Objectives
- Understand overfitting in neural networks
- Implement dropout regularization using TensorFlow/Keras
- Compare model performance with and without dropout
- Analyse training behaviour using visualisations
- Evaluate model generalisation using validation metrics

---

##  Dataset
The project uses the MNIST dataset:

- 60,000 training samples  
- 10,000 testing samples  
- 28×28 grayscale images  
- 10 classes (digits 0–9)


---

##  Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

##  Visualisations
The following visualisations are generated:

1. Accuracy comparison (Dropout vs No Dropout)  
2. Loss comparison  
3. Effect of dropout rates  
4. Overfitting gap analysis  
5. Confusion matrix  

These visualisations provide both performance insights and interpretability.
